# Category Taxonomy — カテゴリ体系の再設計

> ステータス: 確定
> 作成日: 2026-04-18

## 目的

Vector は先端テックのニュース収集を目的とする。
「先端テック」の定義は **(a) ディープテック寄り — 商用化前〜初期の技術領域** とする。
VC・スタートアップ文脈の新興テック全般 (b) や、テック業界全般 (c) は採らない。

カテゴリは「ユーザーが先端テックの動向を領域別に追える」ことを目的とした **粗い大分類** とし、
細粒度の話題分類は別レイヤー (AI トピック層) に委ねる。

## カテゴリ一覧 (10)

| カテゴリ | 主たる成果物 |
|---|---|
| AI | AI モデル・サービス・エージェント・AI 企業動向 |
| ロボティクス・モビリティ | 自律ロボット・自動運転・ドローン・eVTOL・新型乗り物 |
| 半導体 | チップ・製造プロセス・製造装置・パッケージング |
| 次世代コンピューティング | 量子・ニューロモーフィック・光・DNA 計算 |
| 次世代ネットワーク | 6G・Open RAN・AI-RAN・SDN・海底ケーブル・DC 間通信 |
| セキュリティ | 先端暗号・AI セキュリティ・機密計算・分散 ID 暗号技術 |
| 宇宙 | 衛星・ロケット・宇宙探査・軌道インフラ |
| ゲノム・バイオ | ゲノム編集・遺伝子治療・合成生物学・mRNA・AI 創薬 |
| 新素材・先進製造 | 新材料・3D プリント・ナノファブリケーション |
| 次世代エネルギー | 核融合・SMR・新型電池・水素・先進地熱 |

## 各カテゴリのスコープ詳細

### AI

AI 技術そのもの・AI サービス・AI 企業の動向を扱う。
AI が応用先で使われている記事 (AI 創薬・AI 材料探索) は応用先カテゴリ側に振り分ける。

### ロボティクス・モビリティ

自律性を持つ物理システム全般。
ヒューマノイド (Figure, 1X)、自動運転 (Waymo)、ドローン (Skydio)、eVTOL (Joby)、
自律航行船、配送ロボットなど。
「フィジカル AI」「Embodied AI」と呼ばれる領域はここで吸収する。

### 半導体

チップ設計 (NVIDIA, AMD, Intel)、ファウンドリ (TSMC)、製造装置 (ASML, TEL, Applied Materials)、
EUV、先端パッケージング、AI チップ専業 (Cerebras, Tenstorrent) を含む。
AI 推論用ハードウェアもここ。

### 次世代コンピューティング

非古典計算パラダイム。量子コンピューティング、ニューロモーフィック、光コンピューティング、DNA 計算など。
半導体カテゴリとの境界は「チップ・製造プロセス」か「新しい計算モデル」かで切る。

### 次世代ネットワーク

ネットワーク技術そのものの先端動向。
6G、Open RAN、AI-RAN、SDN、SASE、海底ケーブル新技術、データセンター間通信、エッジコンピューティング。

- **対象外:** 通信キャリアの料金プラン・加入者数・決算・端末販売。
  Verizon・NTT・KDDI 等のキャリアが主語の業界動向は除外。
- **他カテゴリへ:** 衛星通信→宇宙、量子通信→次世代コンピューティング (or セキュリティ)、
  シリコンフォトニクス→半導体。

### セキュリティ

先端セキュリティ・暗号技術。

- **含む:** PQC、機密計算、準同型暗号、ゼロ知識証明、
  AI セキュリティ (敵対的攻撃・モデル盗難・プロンプトインジェクション対策・AI 安全性)、
  量子暗号通信・QKD、信頼実行環境 (TEE)、セキュアマルチパーティ計算、
  分散 ID の暗号技術。
- **対象外:** 一般的な脆弱性報告、ランサムウェア被害、
  セキュリティ企業の買収・決算、製品発表 (新ファイアウォール等)。
- **AI カテゴリとの境界:** 攻撃手法・防御技術が主題ならセキュリティ、
  AI モデルそのものが主題なら AI。「主たる成果物」ルールで判定。

### 宇宙

衛星 (Starlink, Kuiper)、ロケット、宇宙探査、軌道インフラ、宇宙太陽光発電。

### ゲノム・バイオ

分子・細胞レベルで生命を編集・設計・解読する先端技術。

- **含む:** ゲノム編集 (CRISPR、塩基編集、プライム編集)、遺伝子治療 (AAV、CAR-T、in vivo)、
  合成生物学、mRNA 技術、AI 創薬基盤 (AlphaFold 系、生成 AI 創薬)、
  シーケンシング新技術、オルガノイド、ニューロテック (BCI)、
  長寿・アンチエイジング研究、培養肉・バイオファブリケーション。
- **対象外:** 既存薬の臨床試験結果・薬価交渉・製薬会社の業績/M&A・
  病院経営・医療制度・栄養学/サプリメント。
- **境界:** AI 創薬基盤はゲノム・バイオ (応用先優先)、AI モデル自体の発表は AI。
  診断 AI 機器はゲノム・バイオ (バイオ応用) または ロボティクス・モビリティ (医療ロボット) に主題で振り分け。

### 新素材・先進製造

物質そのものの発見・設計、および新しい物理的実現プロセス。

- **含む:** 新材料発見 (超伝導、ペロブスカイト、新電池材料、メタマテリアル、AI 材料探索)、
  3D プリント・付加製造 (Carbon, Desktop Metal, Relativity Space)、
  ナノファブリケーション、セルフアセンブリ、新製造パラダイム。
- **対象外:** 一般的な工場自動化、既存量産技術の改良、製造業の業績/M&A。
- **他カテゴリへ:** 半導体製造装置→半導体、電池デバイスの量産化→次世代エネルギー、
  自律型工場ロボット→ロボティクス・モビリティ、培養肉・合成生物製造→ゲノム・バイオ。

### 次世代エネルギー

商用化前または商用化初期の新しい発電・蓄電・送電パラダイム。
核融合、SMR、第 4 世代原子炉、先進地熱、固体電池、フロー電池、
水素 (グリーン水素・燃料電池新方式)、超伝導送電。

- **対象外:** 既存技術の業界動向 (シリコン太陽光価格動向、陸上風力建設、火力発電、現行原子力)、
  電力会社の業績、電気料金。
- **他カテゴリへ:** 素材レベルの新発見 (新電池正極材) は新素材・先進製造、
  AI 制御グリッド最適化は AI、宇宙太陽光発電は宇宙。

## 分類判定ルール

### 単一カテゴリ + 主たる成果物ルール

1 記事は 1 カテゴリに分類する。判定基準は
**「報じられている主たる成果物 (artifact) はどのドメインのものか」** の単一質問。

「使われている技術」ではなく「アウトプット」で判定する。
これにより「AI で○○を発見/開発」系は必ず ○○ 側に振り分けられ、
AI カテゴリが "AI 技術そのもの" に絞られる。

| 例 | 判定 |
|---|---|
| OpenAI の新モデル発表 | AI |
| AI で新素材発見 | 新素材・先進製造 |
| ロシュ x NVIDIA 提携 (AI 創薬基盤) | ゲノム・バイオ |
| Verizon x NVIDIA AI-RAN 実証 | 次世代ネットワーク |
| Verizon の新料金プラン | 対象外 |

### 上流→下流の関係

カテゴリ間には暗黙の上流/下流関係がある:

- 新素材・先進製造 (源泉) → 次世代エネルギー (電池デバイス)、ロボティクス・モビリティ (軽量素材)、半導体 (新プロセス)
- AI (技術) → ゲノム・バイオ (AI 創薬)、新素材・先進製造 (AI 材料探索)
- 半導体 (チップ) → AI (推論ハード)、ロボティクス (組込)

「主たる成果物」ルールはこの構造を扱える。
同じ素材技術でも、報道されるレイヤー (素材発見か応用デバイスかシステム統合か) で分類が変わる。

### 業界動向の除外

「○○社が △△ 億円調達」「□□ サービス開始」「決算発表」「M&A」「規制対応」のような業界動向は、
技術内容が先端テック寄りでない限り **収集対象外** とする。

これは「フィンテック」「通信」を採用しなかった理由と一貫する。
業界カテゴリは、目的 (先端テック収集) と中身 (業界紙化) のずれを生む。

## 命名原則

1. **対象領域そのものを指す名詞で揃える**
   - 「気候テック」のような「目的+手段」型の命名は使わない (他カテゴリと粒度が揃わない)

2. **「次世代◯◯」は既存パラダイムとの明確な対比軸が存在する場合のみ使う**
   - 古典コンピューティング ↔ 量子等、既存通信 ↔ 6G/Open RAN、火力/現行原子力 ↔ 核融合/SMR
   - 対比される既存版がない領域に「次世代」を付けるのは形骸化を招くため避ける
     (「次世代バイオ」を採らなかった理由)

3. **業界カテゴリは避ける**
   - 「フィンテック」「通信」「ライフサイエンス」のような業界 (Industry) ではなく、
     領域 (Domain) で名前を付ける

4. **業界動向を名前で除外できるようにする**
   - 「次世代エネルギー」が電力会社業績を、「次世代ネットワーク」がキャリア料金プランを、
     「先進製造」が一般工場自動化を、それぞれ名前から自然に除外する構造を持つ

## 検討した代替案と却下理由

### 「AI・ML」→「AI」

ML は AI に包含される概念で並列の意味がない。冗長。
他カテゴリ (新素材・先進製造) は異なる概念の並列だが、AI・ML は同じ概念の重複。

### 「半導体・コンピューティング」→「半導体」

「コンピューティング」が次世代コンピューティングと重複し、両カテゴリの境界が曖昧になる。
「半導体」単独でチップ・製造・装置・パッケージングを十分カバーできる。

### 「気候テック」→「次世代エネルギー」

「気候テック」は他カテゴリと命名粒度が異なる (目的+手段型)。
「次世代エネルギー」にすることで対象領域名で揃い、命名原則が一貫する。
CCUS・ジオエンジニアリング等の境界ケースは新素材・先進製造で吸収可能。

### 「ロボティクス・自動運転」→「ロボティクス・モビリティ」

「自動運転」だと地上の車に限定される印象。
「モビリティ」で eVTOL、ドローン物流、自律航行船まで自然に含められる。

### 「次世代ネットワーク」(独立カテゴリ採用)

当初は通信を廃止して他カテゴリに分散吸収する案もあった。
しかし 6G・Open RAN・AI-RAN・SDN・海底ケーブル・データセンター間通信は
他カテゴリに居場所がなく、構造的に取りこぼす。
先端テック収集の目的に対する穴になるため、独立カテゴリとして残す判断。

命名は「ネットワーク」「通信・ネットワーク」も検討したが、
「次世代ネットワーク」が他の「次世代◯◯」と整合し、
キャリア業界動向を名前から除外できるため採用。

### 「ライフサイエンス」→「ゲノム・バイオ」

「ライフサイエンス」は業界全般を指す広い言葉で、業界動向を呼び込みやすい。
命名原則「業界カテゴリは避ける」に違反。

「次世代バイオ」案も検討したが却下。バイオ領域には「明確に対比される既存版」が存在せず
(CRISPR は 20 年以上の歴史があり「次世代」と呼ぶ違和感がある)、
「次世代◯◯」を機械的に適用すると命名原則の形骸化を招く。

「ゲノム・バイオ」は DNA 技術の中心性を伝え、合成生物学・mRNA・ニューロテック等も
並列の「バイオ」で吸収できる。

### 「新素材・製造」→「新素材・先進製造」

「製造」単独は製造業全般を含む広い言葉で、業界動向を呼び込みやすい。
「先進」を加えることで、3D プリント・ナノファブリケーション・セルフアセンブリ等の
先端技術に焦点を絞り、業界動向を名前から除外できる。

### 「フィンテック」「通信」カテゴリの廃止

両者とも業界 (Industry) ベースの命名で、カテゴリ内容が業界紙化する構造的問題があった。

- フィンテック内の先端テック要素 (PQC 金融応用・AI トレーディング) は
  他カテゴリ (セキュリティ・AI) で吸収可能
- 通信内の先端テック要素は次世代ネットワーク・宇宙・次世代コンピューティング・半導体で吸収可能
- ビジネスモデル系 (決済・ネオバンク・BNPL・キャリア料金プラン) は
  先端テック (a) 定義から外れるため収集対象外

## AI トピック層との役割分担

カテゴリ層で粗く分類した後、各カテゴリ配下で AI がトピックを動的に生成して細分化する 2 段構成。

| レイヤー | 役割 | 性質 |
|---|---|---|
| カテゴリ | 領域別の大分類 | 安定 (長期固定) |
| AI トピック | カテゴリ内の話題抽出 | 動的 (新しい話題が浮かび上がる) |

カテゴリを 10 個に絞り、トピック層に細分化を任せることで:

- カテゴリ自体は長期的に触らずに済む
- 新しい話題が現れても自然に拾える
- 既存タグ体系との関係は別途設計

## 既存データの扱い

現状の DB に蓄積されたカテゴリ分類済み記事は **破棄** し、
新カテゴリ体系での新規分析からやり直す。

理由:

- 既存分類は新カテゴリ体系と粒度・命名が異なり、機械的なマッピングが困難
- 旧分類ロジックは「AI 的要素があれば AI・ML」のような侵食パターンを含んでおり、
  再利用すると新カテゴリ体系の品質を下げる
- 一括 AI 再分類のコスト・時間より、新規積み上げの方が結果的に早い

## 未決事項

- **「対象外」記事の扱い:** 10 カテゴリのいずれにも該当しない記事をどう処理するか
  (「該当なし」を許容するか、分析スキップか、プロンプト・バリデーションへの影響)。
  別セッションで検討。
- **次世代ネットワークの記事ボリューム:** 運用開始後に実績を見て、
  独立カテゴリとして維持するか他カテゴリへの統合を検討。

## 旧カテゴリからの変更サマリ

| 旧カテゴリ | 新カテゴリ | 変更内容 |
|---|---|---|
| AI・ML (`ai_ml`) | AI | ML 冗長削除 |
| ロボティクス (`robotics`) | ロボティクス・モビリティ | eVTOL・ドローン等を明示的に包含 |
| 半導体 (`semiconductor`) | 半導体 | 変更なし |
| 量子コンピュータ (`quantum`) | 次世代コンピューティング | ニューロモーフィック・光・DNA 計算を包含 |
| 通信 (`telecom`) | 次世代ネットワーク | キャリア業界動向を除外、先端技術に限定 |
| — (新設) | セキュリティ | 先端暗号・AI セキュリティに特化 |
| 宇宙 (`space`) | 宇宙 | 変更なし |
| バイオテック (`biotech`) | ゲノム・バイオ | 業界動向を除外、技術領域を明確化 |
| 素材科学 (`materials`) | 新素材・先進製造 | 3D プリント等の先進製造を明示的に包含 |
| エネルギー (`energy`) | 次世代エネルギー | 既存技術の業界動向を除外 |
| フィンテック (`fintech`) | — (廃止) | 先端技術要素は他カテゴリで吸収 |
